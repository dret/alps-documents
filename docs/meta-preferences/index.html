Meta-Preferences - Client Preferences for Service Responses

This is a semantic profile for RWMBook. Services SHOULD provide a link relation with the value of 'meta'. Clients MAY follow this link to get a list of all the content variation parameters supported by the service API.

The contents of the resulting representation SHOULD reflect the available options for the current runnin context. For example, logged in users might see different values than anonymous users.
