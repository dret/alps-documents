specifications

  • type: semantic
  • title: List of open specifications
  • doc: List of open specifications adopted as part of operations
  • def: http://apisjson.org/format/apisjson_0.16.txt
  • tag: ontology, group
  • descriptor
    id type title
    name semantic Name of API or collection
    description semantic Description of the collection
    image semantic Web URL leading to associated image
    url semantic Web URL
    version semantic String representing the version number
    tags semantic Array of descriptive strings
    properties semantic A list of related properties
    contact semantic A list of Persons and/or Organizations (vCard)
