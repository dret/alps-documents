Most Recently Used (MRU) Profile

Most Recently Used (MRU) profile for RWMBook. Services SHOULD emit a short list of the most recently-used URLs when clients interact with the service. This list MAY be filtered to reflect the MRUs of this particular client and/or this particular session between client and service.
